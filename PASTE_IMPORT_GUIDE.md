# תכונת "הדבק וייבא" - מדריך מהיר

## מה זה?

תכונה חדשה המאפשרת לייבא רשימות ימי הולדת ישירות מטקסט חופשי - מוואטסאפ, הערות, או כל מקום אחר!

## איך להשתמש?

### שלב 1: לחץ על "הדבק וייבא"
בדשבורד הראשי, תמצא כפתור סגול חדש לצד כפתורי הייבוא הרגילים.

### שלב 2: הדבק את הרשימה שלך
העתק רשימה מכל מקום והדבק באיזור הטקסט. לדוגמה:

```
משה כהן 15/03/1990
שרה לוי 22.05.85 בלילה
דוד ישראלי 1985-12-10 (חבר טוב)
```

### שלב 3: לחץ "נתח רשימה"
המערכת תפרסר אוטומטית את הטקסט ותחלץ:
- שמות (כולל פטרונימיקה עברית)
- תאריכי לידה (במגוון פורמטים)
- לידה אחרי שקיעה (אם צוין)
- הערות (מסוגריים)

### שלב 4: בדוק ואשר
המערכת תפתח את מסך התצוגה המקדימה (כמו ב-CSV), שם תוכל:
- לראות איך המערכת פרסרה את הנתונים
- לתקן שגיאות
- לבחור לאיזו קבוצה להוסיף
- לאשר את הייבוא

## פורמטים נתמכים

התכונה תומכת במגוון רחב של פורמטי תאריכים:

✅ `DD/MM/YYYY` - 15/03/1990
✅ `DD.MM.YYYY` - 15.03.1990
✅ `DD/MM/YY` - 15/03/90
✅ `DD.MM.YY` - 15.03.90
✅ `YYYY-MM-DD` - 1990-03-15
✅ `D/M/YYYY` - 5/3/1990
✅ `D/M/YY` - 5/3/90

## תכונות מיוחדות

### זיהוי "אחרי שקיעה"
פשוט הוסף אחת מהמילים הבאות והמערכת תזהה אוטומטית:
- בלילה
- ערב / בערב
- שקיעה / אחרי שקיעה
- אחה"צ
- PM

**דוגמה:**
```
שרה לוי 22/05/1985 בלילה
```

### הערות בסוגריים
כל טקסט בסוגריים יועבר אוטומטית לשדה ההערות:
```
דוד כהן 10/12/1985 (בן דוד)
```

### תמיכה בשמות עבריים
המערכת מזהה ומסירה אוטומטית כינויים:
- הרב, רב, מר, מרת, גב', ד"ר, פרופ'

**דוגמה:**
```
הרב יוסף מזרחי 5/6/1978
→ שם פרטי: יוסף, שם משפחה: מזרחי
```

### פטרונימיקה (בן/בר)
המערכת מטפלת נכון בשמות עם "בן" או "בר":
```
דוד בן גוריון 1886-10-16
→ שם פרטי: דוד בן, שם משפחה: גוריון
```

## טיפים לשימוש

1. **שורה אחת = רשומה אחת** - כל שורה צריכה להכיל שם ותאריך
2. **לא חובה לעצב** - המערכת תזהה את התאריך בכל מקום בשורה
3. **אפשר לערבב פורמטים** - כל שורה יכולה להיות בפורמט אחר
4. **בדוק לפני אישור** - תמיד עבור על מסך התצוגה המקדימה


## טיפול בשגיאות

אם שורה לא מזוהה:
- ודא שיש תאריך תקין בשורה
- ודא שיש שם (אחרי הסרת התאריך)
- נסה פורמט תאריך אחר

המערכת תראה אזהרות עבור:
- תאריכים עמומים (5/6 - אפריל או יוני?)
- שם משפחה חסר
- כפילויות אפשריות
